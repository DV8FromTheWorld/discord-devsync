
## Knowledge Base

Accumulated knowledge is stored in md files in `~/discord-kb/` including:
- System architecture overviews
- Key file/directory locations
- API patterns and integration points
- Database schema relationships

To see available topics, `ls -R ~/discord-kb`, or use grep to fulltext search this directory.

Before beginning a new task, always check the KB for relevant info. After finishing a task, consider adding to the KB.
**Code is the source of truth.** If you ever find outdated information in the knowledge base, pause and update it!
When updating the knowledge base, prioritize durable, reusable knowledge; elide one-time info such as the details of
a particular bugfix.
